Scholar Manager

Linux

Pour pouvoir tester l'application il faut suivre les dépendences suivantes :

ouvrir un terminal

installer mongodb en tapant les commandes suivantes : 

$ echo "deb http://repo.mongodb.org/apt/debian wheezy/mongodb-org/3.2 main" | sudo tee /etc/apt/sources.list.d/mongodb-org-3.2.list
$ sudo apt-get update
$ sudo apt-get install -y mongodb-org

installer curl par la commande suivante:

$ sudo apt-get install curl

installer nodejs par les commande suivantes:

$ curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash -
$ sudo apt-get install -y nodejs

installer bower : 

$ npm install -g bower
installer git : 

$ apt-get install git

cloner le projet : 

$ git clone https://github.com/messaooudi/PFA2015.git

entrer dans la repertoire du projet : 

$ cd PFA2015

installer les dependances coté serveur : 

$ npm install

creér une repertoire data : 

$ mkdir data

lancer la base de donner mongodb : 

$ mongod -db path data

lancer un autre terminal est deplacer dans la repertoire du projet PFA2015 :

puis deplacer dans la repertoire public : 

$ cd public

installer les dependances front-end : 

$ bower install

revenir au repertoire courant PFA2015 et lancer l'application : 

$ node app.js

l'url de l'application est : localhost:801
